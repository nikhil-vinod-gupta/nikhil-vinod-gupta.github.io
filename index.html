Key Platforms & Projects

Selected platforms and frameworks I have driven at Walmart to reduce toil, improve reliability, and enable teams to move faster.

AlertBeacon

No-code alert lifecycle platform used as a central entry point for creating, discovering, and validating alerts across systems.

  • Template-driven alert creation for stacks like WCNP, Azure SQL, and Kafka.
  • Built-in checks for broken, noisy, or duplicate alerts using incident/notification data.
  • Designed to be reusable across teams without extra onboarding.

DB Inspector

Azure SQL SRE assistant that surfaces bottlenecks and suggests remediation for developers and DBAs.

  • Shows CPU, waits, blocking sessions, and heavy queries via UI and APIs.
  • Opinionated recommendations for indexing, query patterns, and connection behaviour.
  • Integrates with existing monitoring so teams do not need a new tool stack.

DR Playbook Automation

Framework that generates and maintains DR / failover playbooks for hundreds of applications with one click.

  • Pulls metadata from services, clusters, and databases to auto-build playbooks.
  • Keeps documentation in sync with infra changes and decommissions.
  • Makes DR a living asset instead of a one-time exercise.

One Stop Monitoring

Unified observability entry point for Catalog to view health across WCNP, Kafka, Cassandra, Cosmos, Azure SQL, AlloyDB and more.

  • Product / namespace-based navigation instead of per-tool dashboards.
  • Reduces time to understand impact during incidents and peak events.
  • Designed so teams can onboard with almost no friction.
